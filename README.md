# Raycast — Canva Pro ราคาถูก ระดับพรีเมียม

## โลโก้แบรนด์ (รูปเอง)

1. วางไฟล์รูปในโฟลเดอร์ **`public/`** เช่น **`public/logo.png`**
2. ในเว็บอ้างที่ **`/logo.png`** เท่านั้น (ไม่มี path `@public/` ใน URL)
3. **commit ไฟล์เข้า git** — บน Railway/Linux ตัวพิมพ์เล็ก–ใหญ่ต้องตรง (เช่น `logo.png` ไม่ใช่ `Logo.png`)
4. (ทางเลือก) ใน `.env` หรือ Railway Variables:

   ```env
   NEXT_PUBLIC_BRAND_LOGO_URL="/logo.png"
   ```

   หรือ URL ภายนอก เช่น `https://example.com/my-logo.png`  
   ถ้าไม่ตั้งค่า แอปจะใช้ **`/logo.png` เป็นค่าเริ่มต้น** (ไม่พึ่ง build-time env — แก้กรณี deploy แล้วโลโก้ไม่ขึ้น)

## ทำไมต้อง Raycast

- **Canva Pro เต็มรูปแบบ** — เข้าถึง Brand Kit, Templates และฟีเจอร์พรีเมียมทั้งหมด
- **ราคาถูกกว่า** — เริ่มต้นเพียง **15 บาท/7 วัน** (เฉลี่ย 2.1 บาท/วัน)
- **รวดเร็ว** — เปิดใช้งานได้ภายใน 5–30 นาที หลังยืนยันการชำระเงิน
- **ชำระเงินง่าย** — สแกน PromptPay QR จ่ายได้ทันที ไม่ต้องกรอกข้อมูลบัตร
- **แนะนำเพื่อนรับส่วนลด** — แชร์ลิงก์ให้เพื่อน ทั้งคู่ได้ส่วนลดทันที
- **ติดตามวันหมดอายุได้** — ดูเวลาที่เหลือในแดชบอร์ด พร้อมแจ้งเตือนล่วงหน้า
- **รองรับ 24/7** — ทีมงานพร้อมช่วยเหลือทุกวัน

## คำถามที่พบบ่อย (FAQ)

**Q: Canva Pro ที่ได้ เป็นบัญชีจริงไหม?**
A: ใช่ เราจัดหาบัญชี Canva Pro ให้คุณโดยตรง พร้อมอีเมลยืนยันเข้าใช้งาน

**Q: วันหมดอายุแล้วยังใช้ได้ไหม?**
A: ไม่ได้ ระบบจะหยุดเข้าถึงทันทีเมื่อหมดอายุ แต่คุณสั่งซื้อต่ออายุได้ตลอด

**Q: จ่ายเงินแล้วใช้งานได้เลยไหม?**
A: แอดมินจะตรวจสอบสลิปภายใน 5–30 นาที ถ้าไม่ติดขัดจะอนุมัติทันที

**Q: ถ้าไม่พอใจมีข้อตกลงอย่างไร?**
A: ติดต่อทีมงานภายใน 7 วันหลังสั่งซื้อ หากพบปัญหาเราจะดูแลให้

**Q: ล็อกอิน Google ขึ้น 403 / disallowed_useragent?**
A: เป็นข้อจำกัดของ Google — ไม่อนุญาต OAuth จากเบราว์เซอร์ในแอป (เช่น LINE, Facebook) ให้เปิดลิงก์เว็บใน **Safari** หรือ **Chrome** แล้วลองใหม่ หรือใช้ล็อกอินด้วยอีเมลแทน
